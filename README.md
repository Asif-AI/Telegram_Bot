# Telegram_Bot using GenAI

## How to run?
## Step 1
# Create a conda environment
```bash
conda create -p ./env python=3.10 -y
```
## Step 2
# Activate the conda environment
```bash
conda activate ./env
```

## Step 3
# Install the requirements
```bash
pip install -r requirements.txt
```
